Useful Redstone Machine Challenge

Create a redstone machine that performs a useful action automatically or when triggered.

Challenge outline

  1. Choose the purpose of your machine.
  2. Build the main structure.
  3. Add redstone components to make it work.
  4. Test the machine several times.
  5. Improve reliability and make the design neat.
Evidence: Screenshot of the machine and, if useful, the redstone circuit.

Success checklist

Stretch challenge

Hide the wiring or add an on/off switch, indicator light or reset button.

Before moving on

  • Check every item on the success checklist.
  • Take your screenshot or prepare to show your build.
  • Make one improvement to make your build clearer, safer, more creative or more detailed.
