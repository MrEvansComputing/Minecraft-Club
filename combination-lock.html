Combination Lock Challenge

Create a door or chest area that only opens when the correct switch combination is used.

RedstoneSecurityProblem solving
Previous Next Mission Select

Challenge outline

  1. Build a door or protected area.
  2. Create a set of levers or buttons as the input.
  3. Connect redstone so only the correct combination works.
  4. Test different combinations.
  5. Add signs or clues if you want players to solve it.
Evidence: Screenshot of the input area and protected door/chest.

Success checklist

Stretch challenge

Hide the mechanism and create a clue puzzle for the correct code.

Before moving on

  • Check every item on the success checklist.
  • Take your screenshot or prepare to show your build.
  • Make one improvement to make your build clearer, safer, more creative or more detailed.
